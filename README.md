# Mermaid Discord Bot
A self-hosted Discord bot used to render diagrams generated by Mermaid.js using its syntax.

## Overview
This bot's only purpose is to allow users to render Mermaid.js in Discord. This is good for people who are lazy to use a renderer outside of Discord.

This bot uses Mermaid.ink as the image URL of the generated diagram.

##Configuration
Create a .env file or set environment variables following the contents of .env.example:
```
# Your discord bot token
DISCORD_TOKEN=
```
